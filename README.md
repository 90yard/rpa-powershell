# PowerShell 업무 자동화 스크립트

보안 운영 업무에서 반복되는 작업을 자동화하기 위한 PowerShell 스크립트 모음입니다.

---

## 스크립트 목록

### 1. 브리핑 자동 입력 (`automate-brefing.ps1`)

당직자 명단을 기반으로 브리핑 보고 내용을 자동으로 입력하는 스크립트입니다.

**기능:**
- 텍스트 파일에서 보고자, 당직자 명단 자동 로드
- 마우스 좌표 이동 및 클릭 자동화 (Win32 API)
- 키보드 입력으로 보고 내용 자동 전송

**사용하는 데이터 파일:**
- `보고자.txt` — 현 시각 보고자
- `금일 야간 당직자.txt` — 금일 야간 당직자 명단
- `내일 주간 당직자.txt` — 내일 주간 당직자 명단
- `내일 야간 당직자.txt` — 내일 야간 당직자 명단
- `내일 주간 근무자.txt` — 내일 주간 근무자(포함) 명단

---

### 2. 프로세스 모니터 (`process-monitor.ps1`)

특정 프로세스의 실행 상태를 감시하고, 종료 시 알림을 표시하는 스크립트입니다.

**기능:**
- 지정한 프로세스를 주기적으로 상태 확인 (기본 5초 간격)
- 프로세스 종료 감지 시 Windows 메시지 박스 알림
- 모니터링 대상 및 확인 주기 파라미터로 설정 가능

**사용 예시:**
```powershell
# 기본 사용 (notepad 모니터링, 5초 간격)
.\process-monitor.ps1

# 함수 직접 호출 (프로세스명, 확인 주기 지정)
Monitor-Process -ProcessName "chrome" -CheckInterval 10
```
